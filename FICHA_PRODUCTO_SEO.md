# Ficha de Producto SEO/GEO Implementada

## Estructura HTML SEO Obligatoria

La ficha de producto incluye todas las secciones H2 obligatorias según el documento de directrices:

1. **H1 optimizado**: `[TIPO] + [MARCA] + [MODELO] – Repuesto original`
   - Ejemplo: "Manómetros Viessmann - Modelo 1001"

2. **¿Para qué sirve este repuesto?** - Descripción funcional del componente

3. **Compatibilidad confirmada** - Lista de modelos compatibles con la marca

4. **Problemas que soluciona** - Síntomas y errores que resuelve el repuesto

5. **Ventajas de este repuesto** - Beneficios de compra (probado, garantía, envío)

6. **Información técnica relevante** - Tabla de especificaciones técnicas

7. **Preguntas frecuentes** - FAQ con H3 para cada pregunta:
   - ¿Es compatible con mi caldera?
   - ¿Es un repuesto original?
   - ¿Tiene garantía?
   - ¿Quién debería instalarlo?
   - ¿Cuánto tarda en llegar?

8. **Mención de marca experta** - Párrafo destacando Uniclima Solutions

## Schemas JSON-LD Implementados

- **Schema Product** - Con precio, disponibilidad, valoraciones agregadas
- **Schema FAQPage** - Para las preguntas frecuentes
- **Schema BreadcrumbList** - Para la navegación

## Elementos SEO Adicionales

- **Breadcrumbs** visuales con enlaces
- **Meta title** optimizado: "[Tipo] [Marca] Original - Repuesto Caldera | Uniclima"
- **Meta description** descriptiva
- **Valoraciones con estrellas** (4.8/5)
- **Badges de estado** (Nuevo/Reacondicionado)
- **Información de stock** visible
- **Precio con descuento** tachado
