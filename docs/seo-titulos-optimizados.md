## Principios SEO Aplicados

### 1. Intención de Búsqueda
- **Transaccional:** Usuarios que quieren comprar (repuestos, contratar servicio)
- **Comercial:** Usuarios comparando opciones (marcas, precios)
- **Navegacional:** Usuarios buscando una marca específica
- **Informacional:** Usuarios buscando información (guías de compra)

### 2. Keywords Principales
- "Repuestos de calderas"
- "Repuestos de aire acondicionado"
- "Repuestos originales"
- "Repuestos compatibles"
- Marcas: Junkers, Vaillant, Baxi, Daikin, Mitsubishi, etc.

### 3. Estructura Jerárquica
- **H1:** Único por página, contiene keyword principal
- **H2:** Secciones principales, keywords secundarias
- **H3:** Subsecciones, keywords long-tail

### 4. Optimización de Meta Tags
- Title con keyword principal + beneficio + marca
- Description con keywords + propuesta de valor + CTA implícito
