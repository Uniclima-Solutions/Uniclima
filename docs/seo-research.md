# Investigación de Intenciones de Búsqueda - Uniclima

## Tipos de Intención de Búsqueda

### 1. Informativa
El usuario busca información sobre un tema. No está listo para comprar.
- Ejemplos: "qué es una placa electrónica de caldera", "cómo funciona un intercambiador de calor"

### 2. Transaccional
El usuario está listo para comprar y busca precios o productos.
- Ejemplos: "comprar placa caldera Junkers", "precio repuesto válvula gas Vaillant"

### 3. Comercial (Investigación de compra)
El usuario investiga antes de comprar, compara opciones.
- Ejemplos: "mejores repuestos calderas", "opiniones repuestos Uniclima"

### 4. Navegacional
El usuario busca una marca o sitio específico.
- Ejemplos: "Uniclima repuestos", "tienda repuestos calderas online"

## Estructura de Títulos Recomendada

### Title Tag (Meta Title)
- Formato: [Keyword Principal] | [Beneficio/Diferenciador] | Uniclima
- Longitud: 50-60 caracteres
- Ejemplo: "Repuestos de Calderas Online | Envío 24h | Uniclima"

### H1 - Título Principal
- Debe contener la keyword principal
- Único por página
- Claro y descriptivo
- Ejemplo: "Repuestos de Calderas: Más de 3.000 Referencias Originales y Compatibles"

### H2 - Secciones Principales
- Deben incluir keywords secundarias
- Responder a preguntas del usuario
- Ejemplos:
  - "Categorías de Repuestos de Calderas"
  - "Marcas de Calderas Disponibles"
  - "¿Por qué elegir Uniclima?"

### H3 - Subsecciones
- Keywords long-tail
- Detalles específicos
- Ejemplos:
  - "Placas Electrónicas para Calderas Junkers"
  - "Intercambiadores de Calor Originales"

### H4 - Detalles Adicionales
- Información complementaria
- FAQs
- Características específicas
